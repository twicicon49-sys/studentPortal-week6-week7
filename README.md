
# StudentPortal (Week 6 & 7)

This project combines CRUD operations and authentication using PHP and MySQL.

## Features
- User Registration
- User Login
- Secure Session Handling
- Dashboard Access Control
- Logout System

## Tech Stack
- PHP
- MySQL
- HTML/CSS
- XAMPP

## How to Run
1. Start Apache and MySQL in XAMPP
2. Import your database in phpMyAdmin
3. Open in browser:
   http://localhost/StudentPortal
